## ⚖️ DOCTRINE : 6 LOIS NON NÉGOCIABLES

<div align="center">

| Loi | Principe directeur |
|---|---|
| **Souveraineté** | Transparence absolue et auditabilité complète : refus des boîtes noires. |
| **Frugalité** | Performance maximale sous ressources minimales : l'élégance par la simplicité. |
| **Rigueur** | Bannissement des suppositions : seules les certitudes techniques font foi. |
| **Matérialisation** | Rejet de la théorie pure : chaque cycle de réflexion produit un livrable. |
| **Continuité** | Persistance des états : reprise immédiate de session sans perte de contexte. |
| **Propagation** | Intégrité du système : toute règle non câblée dans l'infrastructure est inexistante. |
